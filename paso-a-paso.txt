1. Crear carpeta del proyecto, puede ser usando "mkdir" desde la terminal, o simplemente crear una carpeta de la manera comun desde el explorador de archivos

2. Iniciar git desde la carpeta del proyecto, primero hay que abrir el cmd o git bash, desde la ubicacion de la carpeta y luego desde la terminal, se usa el comando "git init"

3. Abrir la carpeta del proyecto con VS Code o cualquier editor de texto, y editar los archivos necesarios

4. Agregar los archivos a git, desde la terminal, con la ubicacion de la carpeta del proyecto, usar el comando "git add (el nombre del archivo)" o si se necesitan subir todos los archivos seria "git add ."

5. Hacer commit a git, desde la terminal, con la ubicacion del proyecto usar el comando "git commit -m 'el mensaje que quiera que aparesca'"

6. Crear repositorio en github.com con el nombre del proyecto que necesite y copiar el comando "git remote add origin (nombre-del-proyecto)"

7. pegar el comando de "git remote add origin" en la terminal, finalmente, usar el comando git push -u origin master, y con eso ya quedaria subido en el repositorio en github.com con el nombre del proyecto
