1. Crear carpeta del proyecto, puede ser usando "mkdir" desde la terminal, o simplemente crear una carpeta de la manera comun desde el explorador de archivos

2. Iniciar git desde la carpeta del proyecto, primero hay que abrir el cmd o git bash, desde la ubicacion de la carpeta y luego desde la terminal, se usa el comando "git init"

3. Abrir la carpeta del proyecto con VS Code o cualquier editor de texto, y editar los archivos necesarios
